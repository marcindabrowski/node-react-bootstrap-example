## Usage

### Install

```bash
npm install
```
or 

```bash
yarn install
```

### bundle dev mode

```bash
npm run dev
```
or 

```bash
yarn dev
```

### dev : hot reload mode

```bash
npm run start
```
or 

```bash
yarn start
```

### tests

```bash
npm run test
```
or 

```bash
yarn test
```

### bundle production mode


```bash
npm run prod
```
or 

```bash
yarn prod
```

### mini node-express server

with server hot reload (*and dev bundle*):
```bash
npm run serve-dev
```
or 

```bash
yarn serve-dev
```

without hot reload (*and prod bundle*):
```bash
npm run serve-prod
```
or 

```bash
yarn serve-prod
```
